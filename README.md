# SLOGAN_GEN_LARA

SLOGAN_GEN_LARA is a beginner-friendly Python command-line application that generates marketing slogans using Azure OpenAI.

## Features
- Generate marketing slogans for any product
- Select target audience and tone
- Choose from multiple slogan styles:
  - General
  - Creative
  - Bold

## Requirements
- Python 3.8 or higher
- Azure OpenAI account

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bryanjohn05/SLOGAN_GEN_LARA.git
   cd SLOGAN_GEN_LARA
2. Install dependencies:
    ```bash
    pip install -r requirements.txt


3. Create a .env file and add your Azure OpenAI credentials:
    ```env
    AZURE_OPENAI_API_KEY=your_api_key_here
    AZURE_OPENAI_ENDPOINT=your_endpoint_here

## Usage

1. Run the program:

```bash
   python main.py
